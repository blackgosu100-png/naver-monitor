개인정보처리방침

1. 수집하는 정보

네이버 경쟁사 모니터링은 서비스 제공을 위해 다음 정보를 처리할 수 있습니다.

  • 회원가입 및 로그인에 필요한 이메일 주소와 인증 정보
  • 사용자가 입력한 웹앱 서버 주소
  • 사용자가 등록한 네이버 스마트스토어 또는 브랜드스토어 상품 URL
  • 상품 페이지에서 조회한 옵션명, 옵션별 재고 수량, 상품 이미지 URL, 조회 시각
  • 확장 프로그램의 로그인 상태와 조회 진행 상태

2. 정보 사용 목적

수집된 정보는 아래 목적에만 사용됩니다.

  • 사용자 계정 로그인 및 세션 유지
  • 사용자가 등록한 경쟁사 상품의 재고 조회
  • 조회 결과 저장 및 대시보드 표시
  • 확장 프로그램과 웹앱 간 정상적인 데이터 동기화

3. Chrome 확장 프로그램 권한 사용

확장 프로그램은 서비스 기능 제공에 필요한 범위에서만 Chrome 권한을 사용합니다.

  • cookies: 사용자의 네이버 로그인 세션 상태를 확인하고 상품 페이지 접근 가능 여부를 확인하기 위해 사용합니다.
  • tabs: 사용자가 등록한 상품 페이지를 열고 조회 중인 탭의 상태를 확인하기 위해 사용합니다.
  • scripting: 사용자가 재고 조회를 요청했을 때 상품 페이지에서 재고 정보를 읽기 위해 사용합니다.
  • storage: 서버 주소, 로그인 토큰, 조회 진행 상태를 사용자의 브라우저에 저장하기 위해 사용합니다.
  • host permissions: 네이버 상품 페이지와 사용자가 로그인한 모니터링 웹앱 서버에 접근하기 위해 사용합니다.

4. 제3자 제공 및 판매

네이버 경쟁사 모니터링은 사용자 데이터를 판매하지 않습니다. 또한 서비스의 핵심 기능 제공과 관련 없는 목적으로 사용자 데이터를 제3자에게 제공하지 않습니다.

5. 보관 및 삭제

서비스 데이터는 사용자가 서비스를 이용하는 동안 보관됩니다. 사용자가 계정 또는 데이터 삭제를 요청하면 운영자는 합리적인 기간 내에 관련 데이터를 삭제합니다.

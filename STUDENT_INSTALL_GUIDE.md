# 네이버 경쟁사 모니터링 설치 안내

이 도구는 네이버 스마트스토어/브랜드스토어 상품의 재고 변화를 기록해서, 전날 같은 시간과 비교해 판매량을 추정하는 용도로 사용합니다.

## 준비물

- Chrome 브라우저
- 선생님이 안내한 웹앱 주소
- 서비스 로그인용 이메일과 비밀번호
- 크롬 확장프로그램 압축 파일

## 1. 웹앱 로그인

1. 선생님이 안내한 웹앱 주소에 접속합니다.
2. 이메일과 비밀번호로 회원가입 또는 로그인합니다.
3. 로그인 후 대시보드가 보이면 정상입니다.

## 2. 크롬 확장프로그램 설치

1. 선생님이 전달한 `naver-monitor-extension.zip` 파일의 압축을 풉니다.
2. Chrome 주소창에 아래 주소를 입력합니다.

```text
chrome://extensions
```

3. 오른쪽 위의 `개발자 모드`를 켭니다.
4. 왼쪽 위의 `압축해제된 확장 프로그램을 로드`를 클릭합니다.
5. 압축을 푼 폴더 안의 `chrome_extension` 폴더를 선택합니다.
6. Chrome 오른쪽 위 확장프로그램 아이콘에서 `네이버 경쟁사 모니터링`을 고정해두면 편합니다.

## 3. 확장프로그램 로그인

1. Chrome 오른쪽 위의 `네이버 경쟁사 모니터링` 확장프로그램을 클릭합니다.
2. `서버 주소`에 선생님이 안내한 웹앱 주소를 입력합니다.

예시:

```text
https://your-app.up.railway.app
```

3. 웹앱과 같은 이메일과 비밀번호를 입력합니다.
4. `서비스 로그인` 버튼을 클릭합니다.
5. 로그인 성공 메시지가 나오면 준비 완료입니다.

## 4. 경쟁사 URL 등록

1. 웹앱에서 `설정`으로 이동합니다.
2. 경쟁사 이름과 네이버 상품 URL을 입력합니다.
3. `추가` 버튼을 클릭합니다.

지원 URL 예시:

```text
https://smartstore.naver.com/스토어명/products/상품번호
https://brand.naver.com/브랜드명/products/상품번호
```

## 5. 재고 조회 방법

1. 웹앱 대시보드에서 `전체 조회`를 클릭합니다.
2. Chrome 확장프로그램을 클릭합니다.
3. `경쟁사 재고 전체 조회`를 클릭합니다.
4. 조회가 끝나면 웹앱 대시보드에서 새로고침합니다.

개별 경쟁사만 조회하고 싶으면 웹앱의 해당 경쟁사 행에서 `조회` 버튼을 누른 뒤, 확장프로그램에서 대기 중인 조회를 실행합니다.

## 6. 권장 사용 루틴

매일 같은 시간에 조회하는 것이 가장 중요합니다.

예시:

```text
월요일 오전 9시 조회
화요일 오전 9시 조회
수요일 오전 9시 조회
```

이렇게 해야 전날 같은 시간 대비 24시간 판매량을 비교하기 쉽습니다.

## 자주 생기는 문제

### 확장프로그램에서 로그인이 안 됩니다

- 서버 주소가 정확한지 확인합니다.
- 주소 끝에 `/`가 있어도 보통 괜찮지만, 안 되면 지우고 다시 입력합니다.
- 웹앱에서 먼저 같은 계정으로 로그인이 되는지 확인합니다.

### 경쟁사 조회가 실패합니다

- 네이버 상품 URL이 정확한지 확인합니다.
- 네이버 로그인 또는 인증 화면이 뜨면 직접 통과한 뒤 다시 조회합니다.
- 실패한 상품만 개별 조회로 다시 실행합니다.

### 웹앱에는 등록했는데 확장프로그램에서 안 보입니다

- 웹앱 계정과 확장프로그램 로그인 계정이 같은지 확인합니다.
- 확장프로그램에서 다시 로그인합니다.

### 다른 수강생의 경쟁사가 보이나요?

아니요. 계정별로 경쟁사 URL과 조회 결과가 분리됩니다.

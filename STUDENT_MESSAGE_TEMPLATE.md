# 수강생 공지용 메시지

안녕하세요. 네이버 경쟁사 재고 모니터링 도구 설치 안내드립니다.

아래 3가지를 준비해주세요.

1. Chrome 브라우저
2. 웹앱 접속 주소
3. 첨부된 `naver-monitor-extension.zip`

## 설치 순서

1. 웹앱 주소에 접속해서 회원가입 또는 로그인합니다.
2. 첨부된 `naver-monitor-extension.zip` 압축을 풉니다.
3. Chrome 주소창에 `chrome://extensions`를 입력합니다.
4. 오른쪽 위 `개발자 모드`를 켭니다.
5. `압축해제된 확장 프로그램을 로드`를 클릭합니다.
6. 압축을 푼 폴더 안의 `chrome_extension` 폴더를 선택합니다.
7. Chrome 오른쪽 위 확장프로그램 아이콘에서 `네이버 경쟁사 모니터링`을 클릭합니다.
8. 서버 주소, 이메일, 비밀번호를 입력하고 `서비스 로그인`을 누릅니다.

## 사용 순서

1. 웹앱 설정에서 경쟁사 상품 URL을 등록합니다.
2. 웹앱 대시보드에서 `전체 조회`를 누릅니다.
3. 크롬 확장프로그램을 열고 `경쟁사 재고 전체 조회`를 누릅니다.
4. 조회가 끝나면 웹앱을 새로고침해서 결과를 확인합니다.

매일 같은 시간에 조회해야 전날 같은 시간 대비 판매량 비교가 정확합니다.
